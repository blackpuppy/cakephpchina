核心库¶

CakePHP 自带丰富的内置函数和类。这些类和函数尽力涵盖网络应用程序必需的一些最常用的 功能。

通用目的¶

有通用目的的库，在 CakePHP 的很多地方重用。

  • 通用
    • 全局常量和函数
    • App Class
    • Events System
    • Collections

行为¶

行为为模型添加更多的功能。CakePHP 自带一些内置的行为，比如 TreeBehavior 和 ContainableBehavior。

要学习如何创建和使用行为，请阅读 行为 一节。

  • 行为
    • 访问控制列表(ACL)助件
    • Containable
    • 翻译行为
    • 树行为

组件¶

CakePHP 有一组组件(component)，帮助处理控制器中的基本任务。欲知如何配置和使用组件， 请看 组件 一节。

  • 组件
    • Pagination
    • Flash
    • Sessions
    • Authentication 认证
    • Form tampering prevention
    • CSRF protection
    • 请求处理
    • Cookie
    • 访问控制列表组件

助件¶

CakePHP 包含了一些有助于建立视图的助件。助件帮助创建完整格式的标记代码(markup) (包括表单)，格式化文本、时间和数字，甚至和流行的 JavaScript 库集成。这里汇总了内 置的助件。

请阅读 助件 来了解更多关于助件、助件的 API、以及如何创建和使用 你自己的助件。

  • 助件
    • 缓存助件
    • Flash
    • FormHelper 助件
    • HtmlHelper助件
    • JsHelper
    • NumberHelper
    • Paginator
    • RSS
    • SessionHelper
    • TextHelper
    • TimeHelper

工具¶

除了核心的 MVC 组件，CakePHP 包括一组很棒的工具类，帮助你做各种事情，象网络服务请求、 缓存、日志、国际化，等等。

  • 工具
    • 缓存
    • CakeEmail
    • 目录和文件 Folder & File
    • Hash
    • HttpSocket
    • Inflector
    • Internationalization & Localization
    • 日志记录
    • CakeNumber
    • Router
    • 数据清理 Data Sanitization
    • 安全
    • Set
    • 字符串 String
    • CakeTime
    • Xml
