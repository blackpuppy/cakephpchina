Test shell¶

一旦你已经开始写 Tests 你可以 使用 test shell 来运行他们。

这里有更多的关于test shell的基本使用资料 从命令行运行测试.

2.1 新版功能: 在2.1中增加 test shell ，在2.0中的 testsuite shell 仍旧可用，但使用的是新函数。
